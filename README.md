# LLM Learn

## 项目来源

本项目包含从 [llm-universe](https://github.com/datawhalechina/llm-universe) 仓库获取的学习材料。

原始项目：[datawhalechina/llm-universe](https://github.com/datawhalechina/llm-universe)

项目名称：动手学大模型应用开发

## 原始项目简介

这是一个面向小白开发者的大模型应用开发教程，旨在基于阿里云服务器，结合个人知识库助手项目，通过一个课程完成大模型开发的重点入门。

主要内容包括：
- 大模型简介
- 如何调用大模型 API
- 知识库搭建
- 构建 RAG 应用
- 验证迭代

## 许可证

原始项目为开源项目，用于教学目的。本仓库仅用于个人学习。

## 学习内容

本项目主要包括：
1. LLM 开发入门
2. LLM 开发技巧
3. LLM 应用实例
